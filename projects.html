Projects

Here are my publicly released projects, in varying states of completion. Ordered roughly by date of creation:

Event Simulator (May 2019)

Repository

A simulation module based on Markov chains. Custom simulations can be created/run by specifying states/transitions in a JSON file and Python code in a second file to be executed after each event.

This was a quick project, but a fascinating way to apply Markov chains to something beyond making text. I also experimented with passing functions as arguments and organizing the program in an object-oriented way.

  • Language/Technology: Python
  • Create an unlimited number of states/transitions
  • Assign probabilities that each transition occurs
  • Structured as a reusable Python module

March Madness Bracket Generator (April 2019)

Repository Link to Web App

A web app that allows users to auto-generate an NCAA Tournament bracket based on a selected team statistic, as well as search for information about each team's roster.

This was my creative final project for my Web Programming course, and so I made use of just about all of the material we covered, including databases, client/server-side code, and page design. I also gained valuable experience with deploying/updating web applications onto a hosting service.

  • Language/Technology: JavaScript, HTML, CSS, Node.js, SQL, Heroku
  • See the accuracy of the generated bracket
  • Persistently save/load brackets to the server

Tk-Code (August-September 2018)

Repository

A basic text editor. I spent a lot of time trying to cleanly organize the app into classes (ex: a Document class for open documents, a Menu class for managing the OS's menubar).

While I did not implement a fully featured IDE, the editor is still usable when working with multiple files and simple command-line tasks.

  • Language/Technology: Ruby, Tk
  • Edit multiply files at once using a tabbed view
  • Collapsable tree view of the working directory
  • Simple integrated terminal emulator

Squad Tactical Game (May-July 2018)

Repository

A 2D, top-down, turn-based tactical game based on the X-COM series. The code is a bit messy because I experimented with inheritance and later a component-based structure for the game's entity objects, but most features are implemented.

The most notable portion of the project is my implementation of the A* pathfinding algorithm, which allows the soldiers to move to a clicked location, avoiding any obstacles along the way. One of my most fun (and challenging) projects.

  • Language/Technology: Ruby, Tk
  • Click to move: soldiers will pathfind to selected location.
  • Chance to hit/damage varies with distance, nearby cover
  • Simple CSV format for level creation
  • Simple enemy AI that flanks your soldiers
