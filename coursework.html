Coursework

Here are the relevant courses I have taken during my time at the University of Arizona:

Web Programming (Spring 2019)

This course covered HTML, CSS, and client/server-side JavaScript in depth, while introducing me to databases, cookies/sessions, and basic web security. The class was challenging, but it allowed me to explore the web as a platform to make interactive applications. My creative final project (a web app to generate NCAA tournament brackets and search player stats) can be found here.

Introduction to Computer Programming II (Spring 2019)

This course greatly expanded my knowledge of data structures and general software development. I studied stacks, queues, trees, software testing, exception/error handling, Markov chains, object-oriented programming, recursion, searching, and sorting. One of the most useful courses I've taken on any subject.

Introduction to Computer Programming I (Fall 2018)

This course introduced me to the Python programming language and expanded on the basic skills I had developed through self-learning with more formal names and definitions.
